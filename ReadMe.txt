Решение состоит из 4 проектов:
1. ADV.SumSquares - сайт по расчету суммы квадратов числел
2. ADV.SumSquares.BL - библиотека с бизнес логикой по расчету суммы квадратов чисел
3. ADV.SumSquares.BLTests - Тестовый проект для тестирования бизнес логики расчетов
4. ADV.SumSquares.IntegrationTest - Интеграционный проект для тестирования целиком сайта с бизнес логикой посредством запуска google chrome.
До запуска теста должен быть запущен сайт на http://localhost:5000.

Сайт доступен по адресу: http://localhost:5000
Доступен один контроллер HomeController.

Метод CheckNumber принимет целое число возвращает булевое значение. 
Доступен по адресу, пример вызова http://localhost:5000/Home/CheckNumber?Numbers=34
Служит для проверки введеных данных на сайте. Доступны запросы GET, POST

Метод CalculationSquares принимает массив целых чисел, возращает JSON объект
{ 
history: ["34*34 + 43*43 + 43*43 + 43*43 + 43*43 = 8552"],
total: 8552,
errorMessage : ""
}
Доступен по адресу: http://localhost:5000/Home/CalculationSquares
Доступен запрос POST
Служит для расчета бизнес логики если что то пошло не так возращает ошибку

Метод Number входящих параметров не имеет. 
Доступен запрос GET, возращает HTML страницу браузеру
